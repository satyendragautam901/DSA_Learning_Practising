## 💬 How to Use

1. Clone the repo:
    ```bash
    git clone https://github.com/satyendragautam901/DSA_Learning_Practising.git
   cd DSA_Learning_Practising
    ```

2. Navigate to any topic and open `.cpp` files in your IDE.

3. Compile and run:
    ```bash
    g++ filename.cpp -o output
    ./output
    ```
